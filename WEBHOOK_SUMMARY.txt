HOW TO USE THE WEBHOOK
================================================================================

1. VIA MODAL SDK (Recommended for Python):
   
   import modal
   app = modal.App.lookup("daily-weather-report")
   webhook = app.function("weather_webhook")
   result = webhook.remote()
   print(result)

2. VIA MODAL DASHBOARD:
   
   Visit: https://modal.com/apps/amankonahbright/main/deployed/daily-weather-report
   Click on "weather_webhook" function
   Click "Run" to execute
   View the JSON response

3. LOCALLY (No cloud required):
   
   python src/main.py
   # Creates: data/houston_texas_forecast.json
   # Generates PDF: data/houston_forecast_report.pdf

================================================================================
RESPONSE DATA
================================================================================

Returns a JSON object with:
  • City Location (Houston, Texas)
  • Coordinates (latitude/longitude)  
  • 7-Day Forecast Summary:
    - High/Low Temperatures (°F)
    - Average Temperature
    - Total Precipitation (inches)
    - Average Wind Speed (mph)
  • Timezone Information
  • Fetch Timestamp

Example Response:
{
  "status": "success",
  "data": {
    "city": "Houston",
    "state": "Texas",
    "latitude": 29.76328,
    "longitude": -95.36327,
    "timezone": "America/Chicago",
    "forecast_summary": {
      "high_temp_f": 65.9,
      "low_temp_f": 31.7,
      "avg_high_temp_f": 59.4,
      "days": 7,

      "avg_wind_mph": 9.9
    },
    "fetched_at": "2026-01-17T02:15:30.123456"
  }
}
